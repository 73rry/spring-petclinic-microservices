- On first run of the cluster:
helm repo add ingress-nginx https://kubernetes.github.io/ingress-nginx
helm repo update
helm install ingress-nginx ingress-nginx/ingress-nginx --namespace ingress-nginx --create-namespace

- On the first run of the deployment:
helm install spring-petclinic helm --namespace <namespace> --values <value-file> --create-namespace

- On following runs:
helm upgrade spring-petclinic helm --namespace <namespace> --values <value-file> --create-namespace
